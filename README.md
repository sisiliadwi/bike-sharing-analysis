# Bike Sharing Analysis Dashboard

## Setup Environment 

For Anaconda

```sh
conda create --name main-ds python=3.9
conda activate main-ds
pip install -r requirements.txt
```

For Shell/Terminal

```sh
mkdir proyek_analisis_data
cd proyek_analisis_data
pipenv install
pipenv shell
pip install -r requirements.txt
```

## Run Streamlit
```sh
streamlit run dashboard.py
```
